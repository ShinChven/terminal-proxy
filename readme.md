# terminal-proxy

A terminal tool to run commands behind proxy in terminal.

## use it like

```bash
termproxy curl www.google.coms
```

## install

```bash
npm install -g terminal-proxy
```

## install from source

```bash
npm link
```

## set proxy before using it

```bash
termproxy-set <YOUR_PROXY_URL>
```

## usage

```bash
termproxy <YOUR_COMMANDS>
```

## show proxy

```bash
termproxy-show
```

## unset proxy

```bash
termproxy-set
```
